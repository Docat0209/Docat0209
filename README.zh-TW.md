# 嗨，我是 Docat (Shane)

**全端開發者 | AI 整合應用**

我建構從 AI 到落地的端對端系統 — 語音驅動的 AI Agent、人臉辨識互動機台、行動應用到 POS 系統。我在乎的是把產品交付出去，而不是停在原型階段。

- 資訊管理系 @ **國立雲林科技大學**
- 全國技能競賽第三名 — 行動應用開發
- 目前專注於 **OpenAI Realtime API** Agent、**RAG** 管線與 Raspberry Pi 邊緣部署
